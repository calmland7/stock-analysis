연간 수백~1000%+ 급등한 모멘텀 종목이 펀더멘털은 계속 폭증하는데 단기 주가만 급락(예: 90일 -37.8%)했다면 → 과열 해소(차익실현) 국면. 적극매수(Strong Buy)가 아니라 분할매수(Accumulate)로 판정한다.

**Why:** 급등이 만든 고PBR(SCZ PBR 4.7배, +1103% 급등 후)은 이익 둔화 시 급격한 디레이팅 위험을 안는다. 일괄 매수는 과열 꼭지를 잡을 위험, 외면은 모멘텀 상실 위험 — 분할이 양쪽을 헤지한다.

**How to apply:** 3트랜치 분할 — ① 조정 구간 즉시 일부 ② 명확한 촉매(상장 승격/실적 등) 확정 시 추격 ③ 촉매 후 눌림목 또는 실적 재확인 시 마무리. 전체 평단 기준 손절(-18% 권역) + 마지막 트랜치는 추가 과열 시 보류 옵션. [[pattern-silver-miner-price-leverage]] [[framework-analyst-conflict-weighting]]
