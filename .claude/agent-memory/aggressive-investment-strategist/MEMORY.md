# Aggressive Investment Strategist — Memory Index

## Patterns & Frameworks
- [은 광산주 가격 레버리지 패턴](pattern_silver_miner_price_leverage.md) — 생산량 감소에도 실적 폭증 = 100% 가격 레버리지, 양날의 검
- [급등 후 조정 분할매수 프레임](framework_post-rally-pullback-accumulate.md) — 대급등 후 펀더멘털 견조한 조정은 분할 진입 기회
- [세 분석가 상충 시 가중치 프레임](framework_analyst-conflict-weighting.md) — 업종 중립이 종목 고유 약점인지 거시 공통인지 구분
- [메모리 슈퍼사이클 후발주자 리레이팅](pattern_memory-supercycle-laggard-rerating.md) — 저PER 후발주자(삼성)는 이익+멀티플 더블 엔진, 비대칭 수익
- [밸류에이션 입력 정합성 검증 프레임](framework_valuation-input-consistency-check.md) — 동일종목 PER 상충(LTM vs forward) 시 보수전제·진입가 제동
- [후행 고PER의 forward 정상화 패턴](pattern_trailing-high-per-forward-normalization.md) — PER70배+가 분기 이익폭증으로 forward 15~25배 정상화, 숫자에 겁먹으면 사이클 놓침
- [이중 엔진(레거시 vs 신성장) 패턴](pattern_dual-engine-legacy-vs-growth.md) — 한 기업 내 쇠퇴 레거시 vs 고성장 신사업 양분 시 업종긍정을 성장축에만 가중·포지션 하향
- [진성 고PER vs 후행 착시 판별](pattern_genuine-high-per-not-trailing-illusion.md) — 직전분기 이익성장 세자릿수%면 forward정상화, 두자릿수%면 진짜 프리미엄
- [가격↓인데 PER↑ = 순손실 함정](framework_price-down-but-per-up-net-loss.md) — 하락에도 PER 오르면 EPS훼손이 원인, 손실의 영업라인 위/아래 위치로 진짜 신호 판별
