재무 분석가가 PER 50배+·PBR 10배+ 고평가를 경고할 때, [[pattern-trailing-high-per-forward-normalization]](후행 착시) 패턴을 기계적으로 적용하면 안 된다. 판별 기준은 **직전 분기 이익 성장 속도**다: YoY +1,000%대 폭증(제주반도체)이면 후행 PER 착시 → forward 정상화. 그러나 YoY +30%대의 견조하지만 비폭발적 성장(LS에코에너지 영업익 +31%, 순이익률 4.38%)이면 forward로도 25~35배에 머무는 **진짜 밸류에이션 프리미엄**이다.

**Why:** 2026-06-03 LS에코에너지: PER 51.5배·PBR 10.25배인데 분기 이익 성장이 +31%로 폭발적이지 않아 forward 정상화가 거의 안 됨. 제주반도체(+1,714%)와 정반대 케이스. 둘 다 고PER이지만 동력이 완전히 다르므로, 같은 패턴으로 묶으면 오판한다.

**How to apply:** (1) 고PER 발견 시 먼저 직전 분기 영업익/순익 YoY를 본다 — 세 자릿수%면 forward 정상화 기대, 두 자릿수%면 프리미엄을 액면 그대로 부담으로 인정. (2) 진짜 프리미엄이면 고PBR을 손절 트리거(PBR 8배 이탈)에 직접 반영하고 포지션을 표준(15~20%)보다 하향(10~14%). (3) 단, 업종이 장기 구조적 사이클(경쟁사 수주잔고로 입증)이면 프리미엄을 견딜 시간이 있으므로 방향성은 유지. [[framework-post-rally-pullback-accumulate]]
