세 분석가 의견이 상충할 때(재무 긍정 + 뉴스 긍정 + 업종 중립) 가중치 결정 프레임: 업종 리서처의 중립이 (a) 해당 종목 고유의 약점에서 온 것인지, (b) 거시 변동성·동종업계 공통 규제처럼 종목 무관한 요인에서 온 것인지 구분한다.

**Why:** (b)라면 업종 중립을 디스카운트하고 재무·뉴스에 더 가중. 단, 업종이 지목한 종목 고유 리스크(예: SCZ의 볼리비아 관할권 P/NAV 40~75% 디스카운트)는 목표가·포지션 크기에 직접 반영해야 한다 — 무시하지 않는다.

**How to apply:** 판정 등급은 강세 쪽(재무·뉴스)에 가중하되, 업종이 짚은 고유 리스크는 (1) 포지션 크기 상한 설정(단일국가 리스크 → 비중 15% 미만), (2) 목표가 멀티플 상단 제한으로 반영. 가중치 근거를 리포트에 반드시 명시. [[framework-post-rally-pullback-accumulate]]
