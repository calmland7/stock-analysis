메모리/반도체 슈퍼사이클 초중반 상승 국면에서, 업종 평균 대비 저PER 상태의 후발주자(예: 삼성전자 PER 9~10배 vs 업종 12~15배, HBM 점유율 SK하이닉스 열위)는 (1) 사이클 이익 증가 + (2) 밸류에이션 리레이팅이 동시에 작동하는 더블 엔진 수혜를 받는다. 무차입/저부채 재무가 더해지면 하방 단단·상방 열린 비대칭 구조.

**Why:** 슈퍼사이클 상승기엔 가격·물량 동반 상승으로 후발주자도 수혜. 저PER + 철벽 재무는 가격 피크아웃 시에도 파국 리스크가 낮아 공격적 비중 확대(15~20%)를 정당화. 선두주자 대비 디스카운트가 클수록 리레이팅 여력이 큼.

**How to apply:** 후발주자 고유 약점(HBM 점유율 등)은 무시하지 말고 (a) 고객사 퀄 통과 등 약점 해소 시그널을 추격 트랜치 트리거로, (b) 약점 미해소 시 목표 멀티플 상단 제한으로 반영. 사이클 피크아웃 신호(고정거래가 2개월 연속 하락 반전)는 비중축소 전환 트리거. [[framework-analyst-conflict-weighting]] [[verdict-2026-05-29-samsung-electronics]]
