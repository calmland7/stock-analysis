# PKC (피케이씨 / 001340 / 코스피 화학)

## 2026 Q1 실적 (연결, 2026-05-15 공시 / 5-20 정정)

- 사업부: CA 493억(+8.2%), 소재 113억(+37.5%) → 반도체 소재 고성장
- **당기순이익 -25억(적자전환)**: 영업 양호에도 순손실. 원인 확인됨 → **전환사채(CB) 평가손실 등 비현금성 항목 58억원**이 회계상 반영된 결과. 현금 유출 없음. 즉 영업 펀더멘털 훼손이 아니라 CB 파생상품 평가 효과(주가 상승 시 CB 부채 평가손 발생). 19회 사모 CB 전환가액 6,798원으로 조정 이력.

## 핵심 관찰 / 리스크 패턴
- **유동비율 만성적 100% 미만(33~47%)**: 유동부채 > 유동자산. 단기 유동성 위험 지속. 등급 결정적 감점 요인.
- 부채비율 3년 연속 상승(99.8→171.6%): CAPEX(군산 고순도 염소 50% 증설, 수백억 규모) 확대로 차입 증가.
- 영업이익은 회복(2026 1Q 7.1%)되나 순이익단 변동성 큼(1Q 적자전환) → 영업외/금융비용 점검.
- 모멘텀: 삼성전자 반도체용 고순도 HCl 공급 확대 요청 → 군산 Cl2 연 1,400~1,500t→2,100~2,200t 증설. 비소 불순물 저감으로 첨단공정 품질인증 통과.

## 등급: C (보통) — 2026-06-08 재확인
유동비율(33.7%)·부채비율(171.6%) 취약, 1Q 순손실이 추가 우려. 그러나 매출 성장·영업이익률 회복·반도체 증설 모멘텀이 일부 상쇄. 모니터링 필요.

**적용 메모**: 화학 소재기업은 유동비율이 구조적으로 낮을 수 있으나 PKC는 33% 수준으로 동종 대비도 낮음. 증설 CAPEX 진행 중이라 차입·운전자본 부담 가중 가능 → 분기 순손실 원인(금융비용/영업외)과 차입 만기 구조 추적 권장.
