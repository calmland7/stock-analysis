# PKC (피케이씨 / 001340 / 코스피 화학)

사업: 클로르알칼리(CA) 및 반도체 스페셜티 가스 소재. 반도체향 고순도 소재 비중 확대 중.

## 핵심 관찰 / 리스크 패턴
- **유동비율 만성적 100% 미만(33~47%)**: 유동부채 > 유동자산. 코스피 화학 소재기업 특성상 설비투자·차입금 기반 운영이나 단기 유동성 위험 지속. 등급 산정 시 결정적 감점 요인.
- 부채비율 3년 연속 상승(99.8→171.6%): CAPEX 확대로 차입 증가 추정.
- 2024 영업이익률 급락(7.6→4.0%) 후 2025 소폭 회복. 원가 부담 영향.
- 2025 순이익 급증(반도체 고순도 소재 믹스 개선)으로 PER 정상화.

## 등급: C (보통)
유동비율·부채비율 취약, 그러나 매출 성장세와 2025 수익성 반등이 상쇄. 모니터링 필요.

**적용 메모**: 화학 소재기업은 유동비율이 구조적으로 낮을 수 있으나 PKC는 33% 수준으로 동종 대비도 낮음 → 운전자본/차입 만기 구조 확인 권장.
