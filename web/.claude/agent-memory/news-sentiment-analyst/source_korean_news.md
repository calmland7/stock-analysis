**검색 도구:** `mcp__claude_ai_PlayMCP__NaverSearch-search_news` (sort=date로 최신순, display 크게).

**검색 키워드 노하우:**
- 종목이 상호변경된 경우 신·구 상호 모두 검색 (예: "PKC" + "백광산업"). 영문 약칭(PKC)이 한글(피케이씨)보다 IR/실적 기사 hit가 훨씬 많음.
- "종목명 + 화학/업종 + 지역(군산)" 조합이 IR 기사를 잘 잡음.
- "종목명 + 목표가 + 증권사명" 으로 애널리스트 리포트 분리 수집.
- 너무 좁은 키워드(예: "LiPF6", "전해질")는 0건 나기 쉬움 → 넓게 검색 후 필터링.

**신뢰도 높은 소스(IR/실적):** 뉴시스, 이데일리, 한국경제, etnews, 가스뉴스(gasnews, 화학 전문), amenews(첨단소재 전문), 더일렉(thelec, 반도체/전자 전문). 급등락주는 이투데이(etoday).

**주의:** 지역 신문(전라일보, 새전북 등)·기부/안전점검 같은 기사는 종목 감성에 중립. 신설법인/동명이인(피케이씨 건설·소프트웨어 등) 오래된 기사 다수 → 종목코드 001340 확인 필수.
